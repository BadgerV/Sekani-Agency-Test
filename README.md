* git clone the repository.

```
  $ git clone https://github.com/BadgerV/Sekani-Agency-Test.git
```

- open `Sekani-Agency-Test`.

```
  $ cd Sekani-Agency-Test
```

- install dependencies.
```
 $ npm install
```

- start web app.

```
 $ npm start
```
